eChecker  
Make auto checkin and checkout for eteams  
Author: cahi1l1yn  
Version:0.9  
----------------------------------------------------  

Usage: eChecker.py -i [checkin_time] -o [checkout_time] -c [Cookie]   

-h Print this help  
-i Set checkin time(%hh:%mm)  
-o Set checkout time(%hh:%mm)  
-c Set your cookie
